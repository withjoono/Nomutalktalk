# 산재(산업재해) 대화형 챗봇 추가 완료

## 🏥 추가된 내용

### 1. 산재보험 카테고리 템플릿 (`models/dialogueTemplates.js`)

완전히 새로운 **산재보험** 카테고리가 추가되었습니다!

#### 진단 단계
- **초기 질문**: 6가지 산재 상황 선택지
  1. 업무 중 다쳤어요 (재해 발생)
  2. 산재 신청을 거부당했어요
  3. 산재 승인이 거부되었어요
  4. 산재 보상금이 적어요
  5. 회사가 산재 처리를 막아요
  6. 직업병 의심

- **후속 질문**: 8개의 상세 정보 수집
  - 사고(증상) 발생 날짜
  - 재해 상황 및 경위
  - 다친 부위와 진단명
  - 병원 치료 여부
  - 회사 신고 요청 여부
  - 근무/휴업 상태
  - 4대 보험 가입 여부
  - 목격자/증거 자료

#### 법적 분석 단계
- **관련 법령**: 5개 핵심 조항
  - 산업재해보상보험법 제5조 (업무상의 재해)
  - 제37조 (요양급여)
  - 제52조 (휴업급여)
  - 제57조 (장해급여)
  - 산업안전보건법 제10조 (사업주의 의무)

- **핵심 포인트**: 업무관련성, 통근재해, 직접 신청, 불이익 금지

#### 해결 방안 단계
**5단계 체계적 해결 방법**:

1. **즉시 병원 치료** (최우선!)
   - 산재 지정 병원 또는 일반 병원
   - 나중에 산재 전환 가능
   
2. **회사에 산재 신고 요청**
   - 사고 후 즉시~3일 이내
   - 산재 보험 신고서 제출
   
3. **근로자 직접 산재 신청**
   - 근로복지공단에 직접 신청
   - 회사 협조 없이도 가능
   - 3년 소멸시효
   
4. **산재 승인 후 급여 신청**
   - 요양급여 (치료비 전액)
   - 휴업급여 (평균임금 70%)
   - 장해급여
   
5. **불승인 시 불복 절차**
   - 심사청구 → 재심사청구 → 행정소송
   - 결정일로부터 90일 이내

#### 긴급 조치 사항
🚨 **7가지 즉시 행동**:
- 즉시 병원 치료 (생명 최우선!)
- 사고 현장 사진 촬영
- 목격자 확보
- 진단서 발급
- 사고 경위서 작성
- 3년 소멸시효 주의
- 불이익 조치 시 즉시 신고

#### 후속 질문
- 산재 신청서 작성 도움
- 근로복지공단 연락처 안내
- 산재 승인 기준 및 서류
- 불복 절차 설명
- 회사 분쟁 대응
- 장해 등급 및 보상금
- 직업병 인정 기준

#### 관련 주제
- 업무상 재해 인정 기준
- 통근 재해
- 요양급여/휴업급여/장해급여
- 산재 보험료
- 산재 은폐와 처벌
- 민사 손해배상 병행
- 직업병 인정 사례
- 산재 승인 후 복직

## 🎯 산재 챗봇 주요 특징

### 1. 생명 우선 접근
- 응급 상황 인식 시 즉시 병원 안내
- 치료가 최우선임을 강조
- 나중에 산재 전환 가능 안내

### 2. 회사 비협조 대응
- 회사 거부 시에도 근로자 직접 신청 가능
- 불이익 조치에 대한 법적 보호 안내
- 증거 확보 방법 상세 안내

### 3. 단계별 명확한 가이드
- 5단계 체계적 절차
- 각 단계별 소요 시간, 장단점 명시
- 상황별 최적 방법 제시

### 4. 실무 중심 조언
- 근로복지공단 신청 절차
- 필요 서류 목록
- 불승인 시 불복 방법
- 소멸시효 등 주의사항

## 🚀 사용 예시

### 시나리오 1: 업무 중 부상

**사용자**: "업무 중에 다쳤어요"

**AI (진단)**: "산업재해로 상담하시는군요. 먼저 현재 상태를 확인하고 싶습니다.

다음 중 해당하는 것을 선택해주세요:
1. 업무 중 다쳤어요 (재해 발생)
2. 산재 신청을 거부당했어요
..."

**사용자**: "1번이요. 어제 작업 중에 손가락을 다쳤어요"

**AI (진단 → 추가 질문)**: "힘드시겠어요. 먼저 병원 치료는 받으셨나요? 진단명은 무엇인가요?"

**사용자**: "오늘 병원 갔는데 손가락 골절이래요"

**AI (법적 분석)**: "정보 감사합니다. 법적으로 분석해드리겠습니다...

[분석 내용]
1. 업무상 재해 인정 가능성: 높음
   - 작업 중 발생한 사고로 업무관련성 명확
   - 산업재해보상보험법 제5조 적용
2. 받을 수 있는 급여:
   - 요양급여: 치료비 전액 (병원비, 수술비, 약값 등)
   - 휴업급여: 평균임금의 70% (치료로 근무 못하는 기간)
..."

**AI (해결 방안)**: "구체적인 해결 방법을 안내드리겠습니다...

**1단계: 즉시 병원 치료** ✅ (이미 완료하셨네요!)
**2단계: 회사에 산재 신고 요청**
- 즉시 회사에 산재 처리 요청하세요
- 산재 보험 신고서 작성 협조 요청
..."

### 시나리오 2: 회사 산재 거부

**사용자**: "회사가 산재 신고를 거부해요"

**AI (진단)**: "회사가 비협조적인 상황이시군요. 걱정하지 마세요. 근로자가 직접 신청할 수 있습니다..."

(이후 직접 신청 절차, 필요 서류, 증거 확보 방법 안내)

## 📋 전체 카테고리 목록 (6개)

1. ✅ **임금** - 체불, 퇴직금, 수당
2. ✅ **해고징계** - 부당해고, 징계
3. ✅ **근로시간** - 연장수당, 근로시간 위반
4. ✅ **휴가휴직** - 연차, 출산휴가, 육아휴직
5. ✅ **산재보험** (신규!) - 산업재해, 산재 신청, 보상
6. ✅ **기타** - 범용 템플릿
